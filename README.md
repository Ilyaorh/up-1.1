Автоматизированная информационная система “Учёта и контроля инвентаризации”.
Система предназначена для автоматизированного учёта и контроля инвентаризации.
Система имеет функционал:
•	Добавление данных в таблицу – необходимо нажать  на кнопку “Добавить” и в сплывающем меню ввести нужные данные, Затем нажать кнопку “Сохранить”.

•	Удаление данных – необходимо выбрать пункт, которой нужно удалить, затем нажать  на кнопку “Удалить” и в сплывающем окне подтвердить удаление. 

<img width="867" height="469" alt="image" src="https://github.com/user-attachments/assets/c9a5a566-4d5c-41a4-90af-1719f20ad176" />

•	Изменение данных – необходимо выбрать пункт, которой нужно изменить, затем нажать  на кнопку “Изменить” и в сплывающем меню заменить данные, затем нажать кнопку “Сохранить”.

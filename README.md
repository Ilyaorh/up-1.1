## Автоматизированная информационная система “Учёта и контроля инвентаризации”.

## 1. Описание проекта
Система предназначена для автоматизированного учёта и контроля инвентаризации.

---

## 2. Цель и задачи проекта
**Цель:** Разработка автоматизированной информационной системы, обеспечивающей полный цикл учёта, контроля и анализа движения материальных ценностей.

**Задачи:**
- Анализ предметной области
- Проектирование архитектуры и данных
- Программная реализация
- Отчётность и аналитика

---

## 3. Функциональность
- Добавление данных в таблицу 
- Удаление данных
- Изменение данных
- Сортировка данных
- Поисковая строка
- Обновление вывода данных

---

## 4. Архитектура проекта
Тип архитектуры: Presentation → Data Access → Model

**Технологии:**
- Visual Studio 2022
- Wpf
- C#
- PostgreSQL
- NUnit
